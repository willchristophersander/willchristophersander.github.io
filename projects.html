AI Speech Emotion Recognition System

UVM · Advanced Machine Learning

Built a CRNN architecture (CNN + BiLSTM + Attention) for multilingual speech emotion recognition across eight datasets. Designed the training pipeline, evaluation, and interpretability tools, then deployed real-time inference with Flask.

  • 81.29% accuracy / 83.63% F1 (4-class) and 76.76% accuracy / 82.80% F1 (6-class)
  • Distance-weighted loss using arousal–valence theory and multilingual training
  • PyTorch, Python, Flask, Grad-CAM, attention maps

C++ Canvas Raster Graphics Editor

UVM · Advanced Programming

Built a real-time raster graphics engine with seven tools, multi-layer RGBA buffers, and diff-based undo/redo. Implemented OpenGL rendering with lazy texture updates and software blending, plus image filters via Python C API + NumPy.

  • 60 FPS at 800×600 with lazy CPU→GPU texture updates
  • Custom rasterization and per-pixel alpha compositing
  • C++, OpenGL, ImGui, Python C API, NumPy
